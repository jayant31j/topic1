IoT Applications

Activity : 1

  • What is an IoT application?

    • The term IoT means network of thing or Internet of things. The applications which can transmit data inbetween devices with the usage of internet are IoT applications. They can work on IoT devices.

  • Can you think of some examples?

    • Some of the Examples are :-

    • Smart home applications (Lighting and air conditioning)
    • Tracking devices
    • Security systems
  • What are the main features needed in an IoT application?

    • Accessibility
    • Connectivity
    • Scalability
    • Fast communitation
    • Good integration of software and hardware
